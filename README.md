# dress-up-application
---
Feel free to mix and match your own outfits in my DressUp app
Enjoy!
## Get Started
1. Fork and clone repository.
2. Install dependencies.
3. Checkout to new branch.
4. Run ysrn start to run on local host.

## Technology Used
- ReactNative.js
- Redux
- React Navigation
